STEP 1: CREATE PROJECT

In terminal:

ng new user-app
cd user-app
ng serve
📁 STEP 2: CREATE COMPONENTS

Run these commands:

ng generate component register
ng generate component login
ng generate component profile
📁 STRUCTURE WILL LOOK LIKE
src/app/
 ├── register/
 ├── login/
 ├── profile/
 ├── app.ts
 ├── app.routes.ts


 RUN
ng serve
